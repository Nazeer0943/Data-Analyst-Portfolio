# Data Dictionary

| Column Name | Description | Data Type |
| :--- | :--- | :--- |
| **transaction_id** | Unique ID for each sale | String (Object) |
| **date** | The date of sale (standardized to YYYY-MM-DD) | Datetime |
| **product_category** | The broad type of item (e.g., Electronics) | String |
| **amount_spent** | The revenue from that specific transaction | Float64 |
| **region** | The store location (NORTH, SOUTH, etc.) | String |
